# Database — Neon PostgreSQL

This folder contains all database-related files:

- **Schema definitions** — Table structures, relationships, constraints
- **Migrations** — Database version control scripts
- **Seed data** — Initial/test data for development
- **Connection config** — Neon PostgreSQL connection settings

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Database | PostgreSQL |
| Hosting | [Neon](https://neon.tech) (serverless Postgres) |
| ORM | SQLAlchemy (used in backend) |
